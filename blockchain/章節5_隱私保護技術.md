## 5.1 零知識證明基礎

### 零知識證明協議設計

零知識證明 (ZKP) 允許一方（證明者）向另一方（驗證者）證明某個陳述為真，而不洩露任何額外信息。

#### 三個核心屬性

```
1. 完整性 (Completeness)：
   如果陳述為真，誠實的證明者可以說服誠實的驗證者

2. 可靠性 (Soundness)：
   如果陳述為假，欺詐的證明者無法說服誠實的驗證者
   （除非以可忽略的概率）

3. 零知識性 (Zero-Knowledge)：
   驗證者除了「陳述為真」這個事實外，
   無法獲得任何額外信息
```

#### 圖三著色問題的 ZKP 協議

```python
import hashlib
import random
from enum import Enum

class Color(Enum):
    RED = 0
    GREEN = 1
    BLUE = 2

class GraphColoring:
    def __init__(self, graph, coloring):
        self.graph = graph      # 鄰接表
        self.coloring = coloring  # {node: Color}

class ZKPCommitScheme:
    """Pedersen 承諾方案（簡化版）"""
    def commit(self, value, randomness):
        data = f"{value}:{randomness}".encode()
        return hashlib.sha256(data).hexdigest()
    
    def verify(self, commitment, value, randomness):
        expected = self.commit(value, randomness)
        return commitment == expected

class GraphColoringZKP:
    def __init__(self, graph, secret_coloring):
        self.graph = graph
        self.secret_coloring = secret_coloring
        self.commit_scheme = ZKPCommitScheme()
    
    def prove_round(self):
        """一輪 ZKP 交互"""
        # 步驟 1：隨機打亂顏色（重要！確保零知識性）
        permutation = self._random_permutation()
        shuffled_coloring = {
            node: permutation[color] 
            for node, color in self.secret_coloring.items()
        }
        
        # 步驟 2：對每個節點的顏色做承諾
        commitments = {}
        randomnesses = {}
        for node, color in shuffled_coloring.items():
            r = random.randint(0, 2**256)
            commitments[node] = self.commit_scheme.commit(color.value, r)
            randomnesses[node] = r
        
        return commitments, randomnesses, shuffled_coloring
    
    def reveal_edge(self, edge, randomnesses, shuffled_coloring):
        """揭示指定邊的兩個端點顏色（回應驗證者的挑戰）"""
        u, v = edge
        return {
            u: (shuffled_coloring[u], randomnesses[u]),
            v: (shuffled_coloring[v], randomnesses[v])
        }
    
    def _random_permutation(self):
        colors = list(Color)
        random.shuffle(colors)
        return {orig: new for orig, new in zip(list(Color), colors)}

class Verifier:
    def __init__(self, graph):
        self.graph = graph
        self.commit_scheme = ZKPCommitScheme()
    
    def challenge(self):
        """隨機選擇一條邊作為挑戰"""
        edges = list(self.graph.edges)
        return random.choice(edges)
    
    def verify_round(self, commitments, edge, revelation):
        u, v = edge
        # 驗證承諾一致性
        for node in [u, v]:
            color, r = revelation[node]
            expected = self.commit_scheme.commit(color.value, r)
            if expected != commitments[node]:
                return False
        
        # 驗證邊的兩個端點顏色不同
        color_u, _ = revelation[u]
        color_v, _ = revelation[v]
        return color_u != color_v
```

#### Fiat-Shamir 轉換（互動 → 非互動）

```python
class FiatShamirTransform:
    """將互動式 ZKP 轉換為非互動式（NIZK）"""
    
    def generate_challenge(self, commitment, public_params):
        """用哈希函數模擬驗證者的隨機挑戰（隨機 Oracle 模型）"""
        data = f"{commitment}:{public_params}".encode()
        challenge_hash = hashlib.sha256(data).hexdigest()
        return int(challenge_hash, 16) % self.challenge_space
    
    def prove(self, witness, statement):
        """生成非互動式證明"""
        # 1. 生成承諾
        randomness = random.randint(0, self.field_order)
        commitment = self.commit(randomness, witness)
        
        # 2. 用哈希生成挑戰（代替驗證者互動）
        challenge = self.generate_challenge(commitment, statement)
        
        # 3. 計算回應
        response = (randomness + challenge * witness) % self.field_order
        
        return {'commitment': commitment, 'challenge': challenge, 'response': response}
    
    def verify(self, proof, statement):
        """驗證非互動式證明"""
        expected_challenge = self.generate_challenge(proof['commitment'], statement)
        if expected_challenge != proof['challenge']:
            return False
        
        # 驗證回應
        lhs = self.base ** proof['response'] % self.modulus
        rhs = (proof['commitment'] * (statement ** proof['challenge'])) % self.modulus
        return lhs == rhs
```

#### ZKP 系統比較

| 特性 | zk-SNARKs | zk-STARKs | Bulletproofs |
|------|-----------|-----------|--------------|
| 證明大小 | 最小 (~200B) | 大 (~100KB) | 中等 (~1KB) |
| 驗證時間 | 最快 | 快 | 中等 |
| 可信設置 | 需要 | 不需要 | 不需要 |
| 抗量子 | 否 | 是 | 否 |
| 適用場景 | ZK-Rollups | 大型計算 | Range proof |

> **🔄 技術更新（2025+）**：ZKP 領域在 2024-2025 年有重大突破：
> - **PLONK / UltraPLONK**：通用可信設置（Universal Trusted Setup），無需針對每個電路重新舉行 Ceremony，已成為 ZK-Rollup 主流證明系統（Polygon zkEVM、zkSync Era 採用）。
> - **Halo2**：Zcash 開發，支援遞歸證明且無需可信設置，被 Scroll 等採用。
> - **Nova / SuperNova / HyperNova**：遞歸 SNARK 的新方向，可增量式驗證計算（Incrementally Verifiable Computation），用於 zkVM 和鏈上證明加速。
> - **FHE（全同態加密）**：雖仍處於研究早期，但 Zama、Fhenix 等項目已嘗試將 FHE 引入區塊鏈，實現鏈上加密計算。

## 5.2 zk-SNARKs 實現

### zk-SNARKs 電路設計

zk-SNARKs (Zero-Knowledge Succinct Non-Interactive Arguments of Knowledge) 通過算術電路將計算問題轉換為可證明的多項式方程。

#### 算術電路基礎

```python
from dataclasses import dataclass
from typing import List, Dict, Optional

@dataclass
class Gate:
    gate_type: str  # 'mul' 或 'add'
    left: str       # 左輸入變量名
    right: str      # 右輸入變量名
    output: str     # 輸出變量名

class ArithmeticCircuit:
    def __init__(self):
        self.gates: List[Gate] = []
        self.inputs: List[str] = []
        self.outputs: List[str] = []
        self.constants: Dict[str, int] = {}
        self.field_order = 2**254 + ...  # BLS12-381 曲線的標量域
    
    def add_input(self, name: str):
        self.inputs.append(name)
    
    def add_mul_gate(self, left: str, right: str, output: str):
        """乘法門：output = left × right"""
        self.gates.append(Gate('mul', left, right, output))
    
    def add_add_gate(self, left: str, right: str, output: str):
        """加法門：output = left + right"""
        self.gates.append(Gate('add', left, right, output))
    
    def evaluate(self, inputs: Dict[str, int]) -> Dict[str, int]:
        values = {**inputs, **self.constants}
        
        for gate in self.gates:
            l = values[gate.left]
            r = values[gate.right]
            
            if gate.gate_type == 'mul':
                values[gate.output] = (l * r) % self.field_order
            else:  # add
                values[gate.output] = (l + r) % self.field_order
        
        return {name: values[name] for name in self.outputs}
```

#### R1CS（秩-1 約束系統）

```python
class R1CS:
    """
    將算術電路轉換為 R1CS 約束：
    每個約束形如：(A · w) × (B · w) = (C · w)
    其中 w 是包含所有中間值的 witness 向量
    """
    
    def __init__(self, num_variables):
        self.num_vars = num_variables
        self.A = []  # 左輸入矩陣
        self.B = []  # 右輸入矩陣
        self.C = []  # 輸出矩陣
    
    def add_constraint(self, a_coeffs, b_coeffs, c_coeffs):
        """添加約束 (a·w) × (b·w) = (c·w)"""
        self.A.append(a_coeffs)
        self.B.append(b_coeffs)
        self.C.append(c_coeffs)
    
    def verify(self, witness: List[int], field_order: int) -> bool:
        """驗證 witness 滿足所有 R1CS 約束"""
        for i in range(len(self.A)):
            a_val = sum(self.A[i][j] * witness[j] for j in range(len(witness))) % field_order
            b_val = sum(self.B[i][j] * witness[j] for j in range(len(witness))) % field_order
            c_val = sum(self.C[i][j] * witness[j] for j in range(len(witness))) % field_order
            
            if (a_val * b_val) % field_order != c_val:
                return False
        return True
    
    @classmethod
    def from_circuit(cls, circuit: ArithmeticCircuit):
        """從算術電路自動生成 R1CS"""
        # 收集所有變量（輸入 + 中間值 + 輸出）
        all_vars = ['1'] + circuit.inputs  # '1' 是常數 1
        for gate in circuit.gates:
            if gate.output not in all_vars:
                all_vars.append(gate.output)
        
        var_index = {v: i for i, v in enumerate(all_vars)}
        r1cs = cls(len(all_vars))
        
        for gate in circuit.gates:
            a = [0] * len(all_vars)
            b = [0] * len(all_vars)
            c = [0] * len(all_vars)
            
            if gate.gate_type == 'mul':
                a[var_index[gate.left]] = 1
                b[var_index[gate.right]] = 1
                c[var_index[gate.output]] = 1
            else:  # add：轉換為乘法（乘以常數 1）
                a[var_index[gate.left]] = 1
                a[var_index[gate.right]] = 1
                b[var_index['1']] = 1
                c[var_index[gate.output]] = 1
            
            r1cs.add_constraint(a, b, c)
        
        return r1cs
```

#### Groth16 證明系統

```python
class Groth16Prover:
    """Groth16 zk-SNARK 證明生成（需要 CRS 可信設置）"""
    
    def __init__(self, proving_key, r1cs):
        self.pk = proving_key
        self.r1cs = r1cs
    
    def prove(self, public_inputs: List[int], private_witness: List[int]):
        """
        生成 Groth16 證明：π = (A, B, C)
        每個元素是橢圓曲線點
        """
        full_witness = [1] + public_inputs + private_witness
        
        # 計算多項式並在橢圓曲線上求值
        # 添加隨機化（r, s）確保零知識性
        r = random.randint(0, self.pk.field_order)
        s = random.randint(0, self.pk.field_order)
        
        # A = α + A(τ) + r·δ (在 G1 上)
        # B = β + B(τ) + s·δ (在 G2 上)
        # C = [Σ(public) + Σ(private) + h(τ)·t(τ)] / δ + s·A + r·B - rs·δ
        
        return {
            'A': self.compute_A(full_witness, r),
            'B': self.compute_B(full_witness, s),
            'C': self.compute_C(full_witness, r, s)
        }
    
    def compute_A(self, witness, r):
        # A = Σ witness[i] * pk.A[i] + r * pk.delta_1
        result = self.pk.alpha_1
        for i, w in enumerate(witness):
            result = ec_add(result, ec_mul(self.pk.A1[i], w))
        result = ec_add(result, ec_mul(self.pk.delta_1, r))
        return result

class Groth16Verifier:
    def verify(self, proof, public_inputs, verification_key):
        """
        驗證 Groth16 證明
        條件：e(A, B) = e(α, β) · e(Σpublic, γ) · e(C, δ)
        """
        lhs = pairing(proof['A'], proof['B'])
        
        rhs_1 = pairing(verification_key.alpha_1, verification_key.beta_2)
        
        # 計算公開輸入的線性組合
        public_sum = verification_key.ic[0]
        for i, inp in enumerate(public_inputs):
            public_sum = ec_add(public_sum, ec_mul(verification_key.ic[i+1], inp))
        rhs_2 = pairing(public_sum, verification_key.gamma_2)
        
        rhs_3 = pairing(proof['C'], verification_key.delta_2)
        
        return lhs == gt_mul(rhs_1, gt_mul(rhs_2, rhs_3))
```

## 5.3 隱私幣實現

### 隱私幣協議設計

隱私幣通過密碼學技術隱藏交易金額、發送者和接收者信息。

#### 環簽名 (Ring Signature)

環簽名允許用戶代表一個組（環）簽名，驗證者只知道簽名來自組中某個成員，但不知道是哪個。

```python
import hashlib
import random

class RingSignature:
    def __init__(self, field_order: int):
        self.q = field_order  # 大質數
        self.g = 2            # 生成元（簡化）
    
    def generate(self, message: bytes, private_key: int, 
                 ring_public_keys: list, signer_index: int):
        """
        生成環簽名
        message: 要簽名的消息
        private_key: 真實簽名者的私鑰
        ring_public_keys: 環成員的公鑰列表
        signer_index: 真實簽名者在環中的位置
        """
        n = len(ring_public_keys)
        
        # 生成隨機 k 值
        k = random.randint(1, self.q - 1)
        
        # 從 signer_index+1 開始，循環生成環的其他部分
        s = [0] * n
        e = [0] * n
        
        # 計算 e[signer_index+1]
        r = pow(self.g, k, self.q)
        e_next = self._hash(message, ring_public_keys, r)
        
        # 正向填充環（非簽名者）
        i = (signer_index + 1) % n
        e[i] = e_next
        
        while i != signer_index:
            s[i] = random.randint(1, self.q - 1)
            r_i = (pow(self.g, s[i], self.q) * pow(ring_public_keys[i], e[i], self.q)) % self.q
            e[(i + 1) % n] = self._hash(message, ring_public_keys, r_i)
            i = (i + 1) % n
        
        # 計算簽名者的 s 值（閉合環）
        s[signer_index] = (k - private_key * e[signer_index]) % (self.q - 1)
        
        return {'e0': e[0], 's': s}
    
    def verify(self, message: bytes, signature: dict, ring_public_keys: list) -> bool:
        """驗證環簽名"""
        n = len(ring_public_keys)
        e = signature['e0']
        s = signature['s']
        
        for i in range(n):
            r_i = (pow(self.g, s[i], self.q) * pow(ring_public_keys[i], e, self.q)) % self.q
            e = self._hash(message, ring_public_keys, r_i)
        
        return e == signature['e0']
    
    def _hash(self, message, public_keys, r):
        data = message + b''.join(pk.to_bytes(32, 'big') for pk in public_keys)
        data += r.to_bytes(32, 'big')
        return int(hashlib.sha256(data).hexdigest(), 16) % self.q
```

#### 隱形地址 (Stealth Address)

隱形地址允許發送者為每筆交易生成一次性地址，只有接收者能發現並控制這些地址。

```python
class StealthAddress:
    """
    基於 ECDH 的隱形地址方案（如 Monero 使用）
    """
    
    def __init__(self, curve_params):
        self.G = curve_params.generator  # 曲線生成點
        self.order = curve_params.order
    
    def generate_recipient_keys(self):
        """接收者生成掃描鑰對和花費鑰對"""
        scan_privkey = random.randint(1, self.order - 1)
        spend_privkey = random.randint(1, self.order - 1)
        
        scan_pubkey = ec_mul(self.G, scan_privkey)
        spend_pubkey = ec_mul(self.G, spend_privkey)
        
        return {
            'scan_privkey': scan_privkey,
            'spend_privkey': spend_privkey,
            'scan_pubkey': scan_pubkey,
            'spend_pubkey': spend_pubkey
        }
    
    def send_to_stealth(self, recipient_scan_pubkey, recipient_spend_pubkey):
        """發送者生成一次性地址"""
        # 生成臨時密鑰對
        r = random.randint(1, self.order - 1)
        R = ec_mul(self.G, r)  # 臨時公鑰，廣播到鏈上
        
        # ECDH 共享密鑰
        shared_secret = ec_mul(recipient_scan_pubkey, r)
        shared_hash = hashlib.sha256(point_to_bytes(shared_secret)).digest()
        
        # 一次性地址 = spend_pubkey + hash(shared_secret) × G
        one_time_pubkey = ec_add(
            recipient_spend_pubkey,
            ec_mul(self.G, int.from_bytes(shared_hash, 'big') % self.order)
        )
        
        return R, one_time_pubkey  # R 廣播，one_time_pubkey 作為接收地址
    
    def scan_for_payments(self, scan_privkey, spend_pubkey, R_values_on_chain):
        """接收者掃描鏈上所有 R 值，找出自己的付款"""
        my_payments = []
        
        for R in R_values_on_chain:
            # 計算共享密鑰
            shared_secret = ec_mul(R, scan_privkey)
            shared_hash = hashlib.sha256(point_to_bytes(shared_secret)).digest()
            
            # 計算期望的一次性地址
            expected_pubkey = ec_add(
                spend_pubkey,
                ec_mul(self.G, int.from_bytes(shared_hash, 'big') % self.order)
            )
            
            # 比較是否匹配鏈上的付款地址
            if expected_pubkey == get_output_address(R):
                spend_privkey = (spend_privkey + int.from_bytes(shared_hash, 'big')) % self.order
                my_payments.append({'R': R, 'spend_privkey': spend_privkey})
        
        return my_payments
```

#### 保密交易 (Confidential Transactions)

使用 Pedersen 承諾隱藏交易金額，同時確保金額守恆（無通脹）。

```python
class PedersenCommitment:
    """
    Pedersen 承諾：C = amount × G + blinding × H
    G 和 H 是橢圓曲線上的兩個獨立點
    """
    
    def __init__(self, G, H, order):
        self.G = G        # 主生成點
        self.H = H        # 盲化因子基點（H = hash_to_curve(...)）
        self.order = order
    
    def commit(self, amount: int, blinding_factor: int):
        """承諾金額：C = amount·G + blinding·H"""
        return ec_add(
            ec_mul(self.G, amount),
            ec_mul(self.H, blinding_factor)
        )
    
    def verify_sum(self, input_commitments, output_commitments):
        """
        驗證輸入總額 = 輸出總額（不揭露具體金額）
        若 Σinputs = Σoutputs，則差值承諾 = 0 點（或已知偏移）
        """
        sum_inputs = ec_identity()
        for c in input_commitments:
            sum_inputs = ec_add(sum_inputs, c)
        
        sum_outputs = ec_identity()
        for c in output_commitments:
            sum_outputs = ec_add(sum_outputs, c)
        
        # 若盲化因子之和也相等，則金額守恆
        # 實際上還需要驗證盲化因子的差值知識
        return sum_inputs == sum_outputs

class RangeProof:
    """
    範圍證明：證明承諾的金額在 [0, 2^n) 範圍內
    防止負數金額攻擊（如承諾 -100 ETH）
    """
    
    def prove(self, amount: int, blinding: int, n_bits: int = 64):
        """

    def __init__(self, pedersen, range_proof_system):
        self.pedersen = pedersen
        self.range_proof = range_proof_system
    
    def create_transaction(self, inputs, outputs, fee):
        """
        inputs: [(amount, blinding, )]
        outputs: [(amount, recipient_pubkey)]
        """
        output_blindings = [random.randint(0, self.pedersen.order-1) for _ in outputs]
        
        # 計算輸入盲化因子之和 = 輸出盲化因子之和（確保承諾可以對消）
        input_blinding_sum = sum(b for _, b in inputs) % self.pedersen.order
        last_output_blinding = (input_blinding_sum - sum(output_blindings[:-1])) % self.pedersen.order
        output_blindings[-1] = last_output_blinding
        
        # 生成輸出承諾和範圍證明
        output_commitments = []
        range_proofs = []
        for (amount, _), blinding in zip(outputs, output_blindings):
            commitment = self.pedersen.commit(amount, blinding)
            range_proof = self.range_proof.prove(amount, blinding)
            output_commitments.append(commitment)
            range_proofs.append(range_proof)
        
        # 生成輸入承諾
        input_commitments = [
            self.pedersen.commit(amount, blinding) for amount, blinding in inputs
        ]
        
        return {
            'input_commitments': input_commitments,
            'output_commitments': output_commitments,
            'range_proofs': range_proofs,
            'fee': fee  # 公開的手續費
        }
    
    def verify_transaction(self, tx):
        """驗證保密交易"""
        # 1. 驗證範圍證明（確保金額非負）
        for commitment, range_proof in zip(tx['output_commitments'], tx['range_proofs']):
            if not self.range_proof.verify(range_proof, commitment):
                return False
        
        # 2. 驗證金額守恆（輸入 = 輸出 + 手續費）
        # fee 作為公開承諾加入等式
        fee_commitment = self.pedersen.commit(tx['fee'], 0)  # 手續費盲化因子為 0
        
        all_outputs = tx['output_commitments'] + [fee_commitment]
        return self.pedersen.verify_sum(tx['input_commitments'], all_outputs)
```
