## 7.1 HD 錢包設計

### 階層式確定性錢包設計

HD 錢包 (Hierarchical Deterministic Wallet) 允許從單一種子 (seed) 生成整棵密鑰樹，實現多地址管理和錢包備份簡化。

#### 核心標準

| 標準 | 功能 |
|------|------|
| BIP32 | 定義子密鑰派生規則（HMAC-SHA512） |
| BIP39 | 助記詞生成（12/24個英文單詞）和種子生成 |
| BIP44 | 標準化派生路徑 `m/44'/coin_type'/account'/change/index` |

#### 密鑰派生算法

```python
import hmac
import hashlib
import struct
from base58 import b58encode_check

class HDWallet:
    BITCOIN_SEED_MAGIC = b"Bitcoin seed"
    HARDENED_OFFSET = 0x80000000
    
    def __init__(self):
        self.master_private_key = None
        self.master_chain_code = None
    
    def from_seed(self, seed: bytes):
        """從種子生成主密鑰 (BIP32 Master Key)"""
        I = hmac.new(self.BITCOIN_SEED_MAGIC, seed, hashlib.sha512).digest()
        self.master_private_key = I[:32]   # 前 32 字節 = 私鑰
        self.master_chain_code = I[32:]    # 後 32 字節 = 鏈碼
        return self
    
    def derive_child_key(self, parent_key: bytes, parent_chain_code: bytes, 
                         index: int, hardened: bool = False):
        """派生子密鑰 (CKDpriv)"""
        if hardened:
            index = index | self.HARDENED_OFFSET
            data = b'\x00' + parent_key + struct.pack('>I', index)
        else:
            # 軟派生：使用公鑰
            parent_pubkey = self._privkey_to_pubkey(parent_key)
            data = parent_pubkey + struct.pack('>I', index)
        
        I = hmac.new(parent_chain_code, data, hashlib.sha512).digest()
        
        child_private_key_int = (
            int.from_bytes(I[:32], 'big') + 
            int.from_bytes(parent_key, 'big')
        ) % self.CURVE_ORDER  # secp256k1 曲線階
        
        child_private_key = child_private_key_int.to_bytes(32, 'big')
        child_chain_code = I[32:]
        
        return child_private_key, child_chain_code
    
    def derive_from_path(self, path: str):
        """
        從路徑派生密鑰
        示例路徑：'m/44'/0'/0'/0/0'
        """
        current_key = self.master_private_key
        current_chain = self.master_chain_code
        
        parts = path.split('/')
        assert parts[0] == 'm', "路徑必須以 'm' 開頭"
        
        for part in parts[1:]:
            hardened = part.endswith("'")
            index = int(part.rstrip("'"))
            current_key, current_chain = self.derive_child_key(
                current_key, current_chain, index, hardened
            )
        
        return current_key, current_chain
    
    def _privkey_to_address(self, private_key: bytes) -> str:
        """私鑰 → P2PKH 地址"""
        pubkey = self._privkey_to_pubkey(private_key)  # 33 bytes 壓縮公鑰
        
        # 1. SHA256(public_key)
        sha256_hash = hashlib.sha256(pubkey).digest()
        
        # 2. RIPEMD160(sha256_hash)
        ripemd160 = hashlib.new('ripemd160')
        ripemd160.update(sha256_hash)
        pubkey_hash = ripemd160.digest()
        
        # 3. 添加版本號 (0x00 = 主網)
        versioned = b'\x00' + pubkey_hash
        
        # 4. 校驗和 (double SHA256 前 4 bytes)
        checksum = hashlib.sha256(hashlib.sha256(versioned).digest()).digest()[:4]
        
        # 5. Base58Check 編碼
        return b58encode_check(versioned).decode('ascii')
```

#### BIP39 助記詞

```python
import os
from Crypto.Protocol.KDF import PBKDF2
from Crypto.Hash import SHA512, HMAC

class MnemonicGenerator:
    WORDLIST_SIZE = 2048  # BIP39 單詞表大小
    
    def __init__(self, wordlist: list):
        self.wordlist = wordlist  # 2048 個英文單詞
    
    def generate_mnemonic(self, strength: int = 128) -> str:
        """
        生成助記詞
        strength: 128 位 = 12 個詞，256 位 = 24 個詞
        """
        assert strength in [128, 160, 192, 224, 256]
        
        # 1. 生成隨機熵
        entropy = os.urandom(strength // 8)
        
        # 2. 計算校驗和（熵的 SHA256 前 strength/32 位）
        h = hashlib.sha256(entropy).digest()
        checksum_bits = strength // 32
        checksum = bin(h[0])[2:].zfill(8)[:checksum_bits]
        
        # 3. 連接熵和校驗和，分割成 11 位的組
        bit_string = bin(int.from_bytes(entropy, 'big'))[2:].zfill(strength)
        bit_string += checksum
        
        # 4. 每 11 位映射到一個單詞
        words = []
        for i in range(0, len(bit_string), 11):
            index = int(bit_string[i:i+11], 2)
            words.append(self.wordlist[index])
        
        return ' '.join(words)
    
    def mnemonic_to_seed(self, mnemonic: str, passphrase: str = '') -> bytes:
        """助記詞 → 64 字節種子 (BIP39 PBKDF2)"""
        mnemonic_bytes = mnemonic.encode('utf-8')
        salt = ('mnemonic' + passphrase).encode('utf-8')
        
        # PBKDF2-HMAC-SHA512，2048 次迭代
        seed = PBKDF2(
            mnemonic_bytes, 
            salt, 
            dkLen=64, 
            count=2048,
            prf=lambda p, s: HMAC.new(p, s, SHA512).digest()
        )
        return seed
```

#### BIP44 標準路徑

```
m / purpose' / coin_type' / account' / change / address_index

purpose: 44'（BIP44 標準）
coin_type: 0'（Bitcoin）, 60'（Ethereum）, 195'（TRX）等
account: 帳戶索引（用於隔離不同用途）
change: 0 = 外部地址（收款），1 = 找零地址
address_index: 地址索引

Bitcoin 示例：m/44'/0'/0'/0/0（第一個帳戶的第一個接收地址）
Ethereum 示例：m/44'/60'/0'/0/0（MetaMask 默認路徑）
```

#### 硬化派生 vs 軟派生

| 特性 | 硬化派生 (`'`) | 軟派生 |
|------|-------------|-------|
| 從父公鑰推導子公鑰 | 不可能 | 可能 |
| 安全性 | 更高 | 較低 |
| 用途 | 帳戶層以上 | 地址生成 |
| 指數範圍 | 0 ~ 2^31-1（加 0x80000000） | 0 ~ 2^31-1 |

## 7.2 多重簽名設計

### 多重簽名安全架構

多重簽名要求多個私鑰共同簽署才能完成交易，提高資金安全性。

#### MultiSignatureWallet 基礎

```python
from dataclasses import dataclass
from typing import List, Dict, Set, Optional

@dataclass
class MultiSigTransaction:
    tx_id: str
    to: str
    value: int
    data: bytes
    signatures: Dict[str, bytes] = None  # signer → signature
    executed: bool = False
    
    def __post_init__(self):
        if self.signatures is None:
            self.signatures = {}

class MultiSignatureWallet:
    def __init__(self, signers: List[str], threshold: int):
        """
        signers: 所有授權簽名者的地址列表
        threshold: 需要的最少簽名數 (M-of-N)
        """
        assert threshold <= len(signers), "門檻不能超過簽名者總數"
        self.signers: Set[str] = set(signers)
        self.threshold = threshold
        self.pending_transactions: Dict[str, MultiSigTransaction] = {}
        self.balance = 0
    
    def propose_transaction(self, proposer: str, to: str, value: int, data: bytes = b'') -> str:
        """提案交易"""
        if proposer not in self.signers:
            raise PermissionError("只有授權簽名者才能提案")
        
        tx = MultiSigTransaction(
            tx_id=self._generate_tx_id(to, value, data),
            to=to,
            value=value,
            data=data
        )
        
        self.pending_transactions[tx.tx_id] = tx
        return tx.tx_id
    
    def sign_transaction(self, signer: str, tx_id: str, signature: bytes):
        """簽署待執行的交易"""
        if signer not in self.signers:
            raise PermissionError("非授權簽名者")
        
        tx = self.pending_transactions.get(tx_id)
        if tx is None:
            raise ValueError("交易不存在")
        if tx.executed:
            raise ValueError("交易已執行")
        
        # 驗證簽名
        if not self._verify_signature(signer, tx, signature):
            raise ValueError("無效簽名")
        
        tx.signatures[signer] = signature
    
    def execute_transaction(self, tx_id: str) -> bool:
        """執行已達到門檻的交易"""
        tx = self.pending_transactions.get(tx_id)
        if tx is None or tx.executed:
            return False
        
        if len(tx.signatures) < self.threshold:
            raise ValueError(f"簽名不足：{len(tx.signatures)}/{self.threshold}")
        
        # 驗證所有簽名
        for signer, sig in tx.signatures.items():
            if not self._verify_signature(signer, tx, sig):
                raise ValueError(f"簽名者 {signer} 的簽名無效")
        
        # 執行交易
        if self.balance < tx.value:
            raise ValueError("餘額不足")
        
        self.balance -= tx.value
        tx.executed = True
        self._send_transaction(tx.to, tx.value, tx.data)
        return True
```

#### Schnorr 多重簽名 (MuSig)

MuSig 將多個簽名聚合成單一簽名，在鏈上看起來與普通簽名無異（隱私性更好，更省 Gas）。

```python
class SchnorrMultiSig:
    """
    Schnorr 多重簽名（簡化版）
    基於 secp256k1 橢圓曲線
    """
    
    def __init__(self, curve_params):
        self.G = curve_params.generator
        self.n = curve_params.order
    
    def aggregate_pubkeys(self, pubkeys: List[bytes]) -> bytes:
        """
        MuSig 密鑰聚合：避免 rogue key 攻擊
        agg_pubkey = Σ H(L, Xi) × Xi
        其中 L = H(X1 || X2 || ... || Xn)
        """
        L = hashlib.sha256(b''.join(pubkeys)).digest()
        
        agg_key = ec_identity()  # 橢圓曲線加法單位元
        for pk in pubkeys:
            coef = int(hashlib.sha256(L + pk).hexdigest(), 16) % self.n
            agg_key = ec_add(agg_key, ec_mul(point_from_bytes(pk), coef))
        
        return point_to_bytes(agg_key)
    
    def round1_commit(self, signer_privkey: int):
        """第一輪：每個簽名者生成 nonce 並提交承諾"""
        r = random.randint(1, self.n - 1)  # 隨機 nonce（絕不重用！）
        R = ec_mul(self.G, r)              # nonce 的公鑰
        commitment = hashlib.sha256(point_to_bytes(R)).digest()
        return r, R, commitment  # r 保密，R 和 commitment 廣播
    
    def round2_sign(self, signer_privkey: int, all_pubkeys: list, 
                    nonce_r: int, all_nonces_R: list, message: bytes):
        """第二輪：揭示 nonce 並生成部分簽名"""
        # 聚合 nonce
        agg_R = ec_identity()
        for R in all_nonces_R:
            agg_R = ec_add(agg_R, R)
        
        # 聚合公鑰
        agg_pubkey = self.aggregate_pubkeys([point_to_bytes(pk) for pk in all_pubkeys])
        
        # 挑戰哈希
        e = int(hashlib.sha256(
            point_to_bytes(agg_R) + agg_pubkey + message
        ).hexdigest(), 16) % self.n
        
        # 計算每個簽名者的係數
        L = hashlib.sha256(b''.join(point_to_bytes(pk) for pk in all_pubkeys)).digest()
        my_pubkey = ec_mul(self.G, signer_privkey)
        a_i = int(hashlib.sha256(L + point_to_bytes(my_pubkey)).hexdigest(), 16) % self.n
        
        # 部分簽名 s_i = r_i + e × a_i × x_i
        s_i = (nonce_r + e * a_i * signer_privkey) % self.n
        return s_i
    
    def aggregate_signatures(self, partial_sigs: List[int], all_nonces_R: list):
        """聚合所有部分簽名"""
        agg_R = ec_identity()
        for R in all_nonces_R:
            agg_R = ec_add(agg_R, R)
        
        agg_s = sum(partial_sigs) % self.n
        
        return {'R': agg_R, 's': agg_s}  # 最終 Schnorr 簽名 (R, s)
    
    def verify(self, signature, agg_pubkey, message: bytes) -> bool:
        """驗證聚合 Schnorr 簽名（與單簽名完全相同）"""
        R, s = signature['R'], signature['s']
        e = int(hashlib.sha256(
            point_to_bytes(R) + point_to_bytes(agg_pubkey) + message
        ).hexdigest(), 16) % self.n
        
        # s × G = R + e × agg_pubkey
        lhs = ec_mul(self.G, s)
        rhs = ec_add(R, ec_mul(agg_pubkey, e))
        return lhs == rhs
```

#### Shamir 秘密分享門限簽名

```python
class ShamirSecretSharing:
    """
    Shamir 秘密分享：將私鑰分成 n 份，任意 t 份可以恢復
    t-of-n 門限方案
    """
    
    PRIME = 2**256 - 189  # 大質數（大於 secp256k1 私鑰的最大值）
    
    def __init__(self, threshold: int, total: int):
        self.t = threshold
        self.n = total
    
    def split_secret(self, secret: int) -> List[tuple]:
        """
        將秘密分成 n 份
        隨機生成 t-1 次多項式：f(x) = secret + a1*x + a2*x^2 + ... + a_{t-1}*x^{t-1}
        每份為 (i, f(i))
        """
        coefficients = [secret] + [
            random.randint(0, self.PRIME - 1) for _ in range(self.t - 1)
        ]
        
        shares = []
        for i in range(1, self.n + 1):
            y = sum(coef * pow(i, power, self.PRIME) 
                   for power, coef in enumerate(coefficients)) % self.PRIME
            shares.append((i, y))
        
        return shares
    
    def recover_secret(self, shares: List[tuple]) -> int:
        """
        從 t 個份額中恢復秘密（拉格朗日插值）
        f(0) = Σ y_i × Π_{j≠i} (x_j / (x_j - x_i))
        """
        assert len(shares) >= self.t, f"需要至少 {self.t} 個份額"
        
        shares = shares[:self.t]  # 只取前 t 個
        x_coords = [share[0] for share in shares]
        y_coords = [share[1] for share in shares]
        
        secret = 0
        for i, (xi, yi) in enumerate(shares):
            numerator = 1
            denominator = 1
            
            for j, (xj, _) in enumerate(shares):
                if i != j:
                    numerator = (numerator * (-xj)) % self.PRIME
                    denominator = (denominator * (xi - xj)) % self.PRIME
            
            # 模逆（費馬小定理：a^(-1) = a^(p-2) mod p）
            lagrange_coef = (yi * numerator * pow(denominator, self.PRIME - 2, self.PRIME)) % self.PRIME
            secret = (secret + lagrange_coef) % self.PRIME
        
        return secret
```

#### 多簽應用場景

```python
class MultiSigGovernance:
    """多簽治理：DAO 資金管理"""
    
    TIMELOCK_DELAY = 48 * 3600  # 48 小時時間鎖
    
    def __init__(self, signers, threshold):
        self.wallet = MultiSignatureWallet(signers, threshold)
        self.pending_proposals = {}
    
    def create_treasury_proposal(self, proposer, recipient, amount, description):
        """創建資金撥付提案（含時間鎖）"""
        tx_id = self.wallet.propose_transaction(proposer, recipient, amount)
        self.pending_proposals[tx_id] = {
            'description': description,
            'proposed_at': time.time(),
            'timelock_until': time.time() + self.TIMELOCK_DELAY
        }
        return tx_id
    
    def execute_after_timelock(self, tx_id):
        """時間鎖期過後執行"""
        proposal = self.pending_proposals.get(tx_id)
        if not proposal:
            raise ValueError("提案不存在")
        if time.time() < proposal['timelock_until']:
            remaining = proposal['timelock_until'] - time.time()
            raise ValueError(f"時間鎖未到期，剩餘 {remaining:.0f} 秒")
        
        return self.wallet.execute_transaction(tx_id)
```

## 7.3 硬件錢包安全

### 硬件錢包安全設計

硬件錢包是專用安全設備，私鑰永遠不離開設備，所有簽名操作在設備內部完成。

#### SecureElement 安全晶片

```python
import os
import hashlib
from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.ciphers import Cipher, algorithms, modes

class SecureElement:
    """安全元件：管理私鑰的生命周期"""
    
    MAX_PIN_ATTEMPTS = 3
    PBKDF2_ITERATIONS = 100000
    
    def __init__(self):
        self.encrypted_private_key = None
        self.pin_hash = None
        self.pin_salt = None
        self.remaining_attempts = self.MAX_PIN_ATTEMPTS
        self.is_locked = True
        self.access_policies = {}
    
    def initialize(self, pin: str) -> bytes:
        """初始化安全元件，設定 PIN 並生成密鑰"""
        self.pin_salt = os.urandom(32)
        
        # PIN 哈希（PBKDF2 強化）
        kdf = PBKDF2HMAC(
            algorithm=hashes.SHA256(),
            length=32,
            salt=self.pin_salt,
            iterations=self.PBKDF2_ITERATIONS
        )
        self.pin_hash = kdf.derive(pin.encode())
        
        # 生成私鑰並加密存儲
        private_key = os.urandom(32)
        encryption_key = self._derive_encryption_key(pin)
        self.encrypted_private_key = self._encrypt(private_key, encryption_key)
        
        # 生成備份助記詞（只在初始化時顯示一次）
        mnemonic = self._generate_mnemonic(private_key)
        
        self.is_locked = False
        return mnemonic
    
    def verify_pin(self, pin: str) -> bool:
        """驗證 PIN（含嘗試次數限制）"""
        if self.remaining_attempts <= 0:
            raise SecurityError("設備已鎖定，需要出廠重置")
        
        kdf = PBKDF2HMAC(
            algorithm=hashes.SHA256(),
            length=32,
            salt=self.pin_salt,
            iterations=self.PBKDF2_ITERATIONS
        )
        
        try:
            kdf.verify(pin.encode(), self.pin_hash)
            self.remaining_attempts = self.MAX_PIN_ATTEMPTS  # 重置計數
            self.is_locked = False
            return True
        except Exception:
            self.remaining_attempts -= 1
            if self.remaining_attempts == 0:
                self._factory_reset()  # 自毀私鑰
            return False
    
    def get_private_key(self, pin: str, purpose: str) -> bytes:
        """獲取私鑰（需驗證 PIN + 檢查存取策略）"""
        if not self.verify_pin(pin):
            raise AuthenticationError("PIN 錯誤")
        
        if purpose not in self.access_policies.get('allowed_operations', []):
            raise PermissionError(f"操作 '{purpose}' 未被授權")
        
        encryption_key = self._derive_encryption_key(pin)
        return self._decrypt(self.encrypted_private_key, encryption_key)
    
    def set_access_policy(self, policy: dict):
        """設定存取控制策略"""
        self.access_policies = policy
    
    def _derive_encryption_key(self, pin: str) -> bytes:
        kdf = PBKDF2HMAC(
            algorithm=hashes.SHA256(),
            length=32,
            salt=self.pin_salt + b'encryption',
            iterations=self.PBKDF2_ITERATIONS
        )
        return kdf.derive(pin.encode())
    
    def _encrypt(self, data: bytes, key: bytes) -> bytes:
        iv = os.urandom(12)
        cipher = Cipher(algorithms.AES(key), modes.GCM(iv))
        encryptor = cipher.encryptor()
        ciphertext = encryptor.update(data) + encryptor.finalize()
        return iv + encryptor.tag + ciphertext
    
    def _decrypt(self, encrypted_data: bytes, key: bytes) -> bytes:
        iv = encrypted_data[:12]
        tag = encrypted_data[12:28]
        ciphertext = encrypted_data[28:]
        cipher = Cipher(algorithms.AES(key), modes.GCM(iv, tag))
        decryptor = cipher.decryptor()
        return decryptor.update(ciphertext) + decryptor.finalize()
    
    def _factory_reset(self):
        """緊急抹除所有敏感數據"""
        self.encrypted_private_key = None
        self.pin_hash = None
        self.is_locked = True
        self.remaining_attempts = 0
```

```python
class HardwareDisplay:
    """硬件錢包顯示屏控制"""

            f"到: {tx_details['to'][:10]}...{tx_details['to'][-6:]}",
            f"Gas: {tx_details['gas_fee']} ETH",
            "按確認鍵簽名"
        ])
        return self.wait_button_confirm("確認交易")

class HardwareWallet:
    """硬件錢包完整實現"""
    
    def __init__(self):
        self.secure_element = SecureElement()
        self.display = HardwareDisplay()
        self.hd_wallet = HDWallet()
        self.is_initialized = False
    
    def initialize_device(self, pin: str):
        """初始化錢包（生成密鑰，顯示助記詞）"""
        mnemonic = self.secure_element.initialize(pin)
        
        # 在設備屏幕上顯示助記詞（要求用戶抄寫）
        self.display.show_mnemonic(mnemonic)
        
        # 設定存取策略（只允許簽名操作）
        self.secure_element.set_access_policy({
            'allowed_operations': ['sign_transaction', 'derive_pubkey'],
            'require_user_confirmation': True,
            'max_amount_without_confirmation': 0  # 任何金額都需確認
        })
        
        self.is_initialized = True
    
    def sign_transaction(self, pin: str, tx_data: dict) -> bytes:
        """
        簽名交易
        1. 驗證 PIN
        2. 顯示交易詳情給用戶
        3. 等待用戶按鍵確認
        4. 在安全元件內部簽名
        """
        if not self.is_initialized:
            raise RuntimeError("設備未初始化")
        
        # 驗證 PIN
        private_key = self.secure_element.get_private_key(pin, 'sign_transaction')
        
        # 在屏幕上顯示交易詳情
        if not self.display.show_transaction_details(tx_data):
            raise UserCancelledError("用戶取消了交易")
        
        # 在安全元件內部簽名（私鑰不離開設備）
        signature = self._sign_ecdsa(private_key, tx_data['hash'])
        
        # 清除內存中的私鑰
        private_key = b'\x00' * len(private_key)
        
        return signature
    
    def derive_public_key(self, pin: str, path: str) -> bytes:
        """從 HD 路徑派生公鑰（不需要私鑰離開設備）"""
        private_key = self.secure_element.get_private_key(pin, 'derive_pubkey')
        
        # 載入 HD 錢包
        self.hd_wallet.master_private_key = private_key
        derived_key, _ = self.hd_wallet.derive_from_path(path)
        public_key = self.hd_wallet._privkey_to_pubkey(derived_key)
        
        # 清除私鑰
        private_key = b'\x00' * len(private_key)
        
        return public_key
    
    def restore_from_mnemonic(self, mnemonic: str, pin: str):
        """從助記詞恢復錢包"""
        gen = MnemonicGenerator(load_bip39_wordlist())
        seed = gen.mnemonic_to_seed(mnemonic)
        
        # 驗證助記詞有效性（BIP39 校驗和）
        if not gen.validate_mnemonic(mnemonic):
            raise ValueError("無效的助記詞")
        
        # 從種子重建私鑰並安全存儲
        self.hd_wallet.from_seed(seed)
        private_key = self.hd_wallet.master_private_key
        
        # 存入安全元件
        self.secure_element.initialized(pin, private_key=private_key)
        self.is_initialized = True
    
    def _sign_ecdsa(self, private_key: bytes, message_hash: bytes) -> bytes:
        """ECDSA 簽名（在安全元件內部執行）"""
        from cryptography.hazmat.primitives.asymmetric import ec
        from cryptography.hazmat.primitives import hashes
        
        sk = ec.derive_private_key(
            int.from_bytes(private_key, 'big'),
            ec.SECP256K1()
        )
        signature = sk.sign(message_hash, ec.ECDSA(hashes.Prehashed()))
        return signature
```

#### 硬件錢包安全設計原則

| 原則 | 實現方式 |
|------|---------|
| 私鑰隔離 | 私鑰永不離開安全晶片 |
| 物理確認 | 所有交易需按下物理按鍵確認 |
| PIN 保護 | PBKDF2 強化，多次失敗後自毀 |
| 螢幕顯示 | 防範中間人攻擊（顯示完整接收地址） |
| 固件簽名 | 只接受廠商簽名的固件更新 |
| 防篡改外殼 | 物理入侵檢測，入侵即清除密鑰 |
| 開源固件 | 社區審計（Trezor 採用開源策略） |

> **🔄 技術更新（2025+）**：
> - **MPC 錢包（Multi-Party Computation Wallet）**：不依賴 Shamir 秘密分享的靜態分片，而是透過 TSS（Threshold Signature Scheme）讓多方在不重組私鑰的情況下協同簽名，私鑰從未以完整形式存在。Fireblocks、Coinbase WaaS、Web3Auth 均採用此方案，是企業級托管主流。
> - **帳戶抽象錢包（EIP-4337 Smart Account）**：Gnosis Safe、Biconomy、ZeroDev 等已提供生產就緒的智能錢包，支援：社交恢復（替代助記詞）、Session Key（限制時間/金額的臨時授權）、Passkey（Face ID/指紋）登入，是 Web3 UX 革新的核心方向。
> - **Passkey 整合**：Coinbase Wallet、MetaMask 等主流錢包已開始支援 WebAuthn Passkey，用戶可用生物識別（指紋/臉部）管理密鑰，無需保存助記詞。
